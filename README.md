## ✨ Features

- 🕒 Four time modes: **15s**, **30s**, **60s**, **120s**
- ⌨️ Real-time **WPM** & **Accuracy** tracking
- 🎨 Custom **color picker** to personalize themes
- 💡 Clean UI inspired by Monkeytype
- 🚀 Fast, responsive, and built for modern browsers
